ml_switcheroo.compiler.backends.extras
======================================

.. py:module:: ml_switcheroo.compiler.backends.extras

.. autoapi-nested-parse::

   Extra Backends (Wrappers for Legacy/IR Emitters).

   Provides CompilerBackend implementations for:
   - TikZ
   - HTML
   - LaTeX DSL
   - MLIR (StableHLO)
