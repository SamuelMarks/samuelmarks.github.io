    Structural Rewriting Logic.

    This module provides the StructureMixin, a component of the PivotRewriter responsible for transforming the scaffolding of Python code: Class definitions and Function definitions.

    It implements logic for: 1. Inheritance Warping: Swapping framework base classes (e.g., torch.nn.Module

    â†” flax.nnx.Module).

    1. Signature Transformation: Injecting or stripping framework-specific state arguments (e.g., rngs) in constructors.

    2. Method Renaming: Mapping forward â†” __call__.

    3. Constructor Logic Injection: Ensuring super().__init__() is present when targeting PyTorch.
