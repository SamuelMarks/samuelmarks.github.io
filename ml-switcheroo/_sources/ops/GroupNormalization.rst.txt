GroupNormalization
==================

A GroupNormalization function. Carries out group normalization as described in the paper https://arxiv.org/abs/1803.08494 This operator transforms input according to ``` y = scale * (x - mean) / sqrt(variance + epsilon) + bias, ``` where the mean and variance are computed per instance per group of c...

**Abstract Signature:**

``GroupNormalization(X: Tensor, scale: Tensor, bias: Tensor, epsilon: float, num_groups: int, stash_type: int)``
