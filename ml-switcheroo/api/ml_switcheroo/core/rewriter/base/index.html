    ml_switcheroo.core.rewriter.baseÂ¶

    Base Rewriter Implementation with Alias and Scope Resolution.

    This module provides the BaseRewriter class, which serves as the foundation for the PivotRewriter. It handles: 1. State Management: Tracking the current scope (global vs class vs function)

    to handle stateful variable detection.

    1. Alias Resolution: Tracking import as statements to resolve t.abs back to torch.abs or np.sum to numpy.sum.

    2. Error Reporting: Collecting failures during the AST walk to be bubbled up to the ASTEngine.

    3. Hook Infrastructure: initializing the HookContext used by plugins.
