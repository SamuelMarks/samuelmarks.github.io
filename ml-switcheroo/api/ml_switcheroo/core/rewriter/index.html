    ml_switcheroo.core.rewriterÂ¶

    Rewriter Package.

    This package provides the PivotRewriter class, composed of several mixins to handle specific aspects of the AST transformation: - Structure: Class and Function definitions. - Calls: Function invocations. - Attributes: Attribute access. - Normalization: Argument mapping. - ControlFlow: Loop and branching logic. - Decorators: Decorator handling.
