    AST Visualization Utility.

    This module provides the MermaidGenerator, a LibCST visitor that traverses an Abstract Syntax Tree and converts it into a Mermaid.js Graph diagram. It is used to generate ‚ÄúBefore‚Äù and ‚ÄúAfter‚Äù snapshots of the code structure during transpilation for debugging and documentation.

    The generator applies specific branding colors and handles complex node representation (like dotted imports) robustly.
