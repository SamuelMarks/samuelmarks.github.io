    ml_switcheroo.core.rewriter.patcherÂ¶

    Graph Patcher for AST Surgery.

    This module implements the execution phase of the Graph-Guided Rewriting pipeline. It applies a set of topological mutations (Deletions, Replacements) generated by the GraphDiffer to the concrete syntax tree (CST) of the source code.

    It bridges the gap between the Abstract Logical Graph and the physical source code by using Provenance Tracking to locate the exact CST nodes corresponding to graph nodes, and the Snippet Emitter to generate valid replacement code.
