   Function Invocation Rewriter.

   Orchestrates the rewrite of a Call node (`func(...)`) via:
   1.  Pre-checks (Unwrapping, Plugins, Lifecycle).
   2.  Strategy execution (Infix, Lambda, Standard).
   3.  Post-processing (Casting, Adapters).
