    ml_switcheroo.core.sass.synthesizer¬∂

    SASS Synthesizer and Register Allocator.

    This module provides the ‚ÄúMiddle-End‚Äù logic for the SASS compiler pipeline. It bridges the gap between high-level Abstract Logic (LogicalGraphs/Python AST) and low-level Physical Assembly (Instruction nodes/Registers).

    It contains: 1. RegisterAllocator: Map Symbolic Variables (e.g., ‚Äòx‚Äô, ‚Äòbias‚Äô) to

    Physical Registers (e.g., ‚ÄòR0‚Äô, ‚ÄòR1‚Äô).

    1. SassSynthesizer: - Target Transformation (`from_graph`): Converts a topological logical graph

      into a linear list of SASS instructions. Supports 1:1 opcode mapping via semantics and 1:N expansion via Kernel Macros (e.g. Conv2d loops).

      • Source Transformation (`to_python`): Converts SASS AST nodes back into Python LibCST nodes for high-level analysis or documentation.

    class ml_switcheroo.core.sass.synthesizer.SassSynthesizer(semantics: ml_switcheroo.semantics.manager.SemanticsManager)¬∂

    Bidirectional transpiler component.

    Handles: 1. Forward (Graph -> SASS): Synthesizes Assembly from Logical Graphs.

    Delegates high-level ops (Conv2d, Linear) to Macros, and low-level ops (Add, Mul) to Semantic Opcode Lookup.

    1. Reverse (SASS -> Python): Synthesizes Python AST from Assembly nodes.

    from_graph(graph: ml_switcheroo.core.graph.LogicalGraph) ‚Üí List[ml_switcheroo.core.sass.nodes.SassNode]¬∂

    Converts a LogicalGraph into a list of SASS AST nodes.

    Process: 1. Sorts nodes topologically. 2. Traverses nodes. 3. For each node:

    1. Check if it matches a Macro (e.g. Conv2d). If so, expand kernel.

    2. If not, lookup abstract opcode mapping (e.g. Add -> FADD).

    3. Allocate/Resolve Input Registers.

    4. Allocate Output Register.

    5. Construct Instruction node.

    1. Handles Input nodes by pre-allocating registers (Contract: R0, R1‚Ä¶).

    Parameters:

    graph (LogicalGraph) ‚Äì The input computation graph.

    Returns:

    A structured list of assembly nodes.

    Return type:

    List[SassNode]
