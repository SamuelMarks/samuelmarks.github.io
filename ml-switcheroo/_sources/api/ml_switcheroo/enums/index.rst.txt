   Used to route definitions to specific JSON files in `src/ml_switcheroo/semantics/`.

   .. attribute:: ARRAY_API

      Basic math and array manipulation (e.g. `abs`, `sum`).
      Maps to `k_array_api.json`.

   .. attribute:: NEURAL

      Nural network layers and stateful operations (e.g. `Conv2d`, `Linear`).
      Maps to `k_neural_net.json`.

   .. attribute:: EXTRAS
