# TensorRT Integration (`tensorrt`)

ONNX9000 exports native Python TensorRT builder scripts from ONNX IR, allowing zero-dependency code generation for Nvidia GPUs.

## CLI Usage

```bash
onnx9000 tensorrt my_model.onnx
```

## Interactive Web Demo

Run `onnx9000 serve` and navigate to `/tensorrt`.
