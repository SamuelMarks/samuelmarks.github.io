ONNX GraphSurgeon Replication & Parity Tracker¶

Description¶

This document tracks the complete reimplementation of ONNX GraphSurgeon within the onnx9000 ecosystem. Unlike the original project, our implementation is tightly integrated into a pure-Python, zero-dependency Intermediate Representation (IR). It avoids the massive C++ Protocol Buffers backend entirely. By operating purely on Python dictionaries, lists, and lightweight dataclasses, this GraphSurgeon can modify, prune, fold, and optimize 1GB+ graphs instantaneously within a browser (WASM/Pyodide) or a distributed serverless environment with zero installation overhead.
