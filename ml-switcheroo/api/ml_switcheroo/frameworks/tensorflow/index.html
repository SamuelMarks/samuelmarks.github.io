    ml_switcheroo.frameworks.tensorflowÂ¶

    TensorFlow Framework Adapter.

    This module implements the adapter for TensorFlow (Core & Keras), providing mappings for math operations, neural layers, and IO. It serves as the bridge between abstract operations and tf.* or tf.keras.* APIs.

    It supports: 1. Live Inspection: Scanning installed TensorFlow modules for new operations. 2. Ghost Mode: Loading API signatures from JSON snapshots when TF is not installed. 3. Semantic Definitions: Static mappings for standard Math, NN, and Optimization ops.
