Slice
=====

Extract a sub-array from the input array using start indices and sizes.

**Abstract Signature:**

``Slice(x: Tensor, start_indices: Union[List[int], Tensor], slice_sizes: Union[List[int], Tensor], axes: Optional[List[int]])``
