    Shared Graph Intermediate Representation.

    This module provides the language-agnostic data structures and extraction logic used to convert Python ASTs into a logical graph of operations (Layers) and data flow (Edges).

    Shared by:

    • TikZ Backend (Visualizer)

    • HTML Backend (Grid Layout)

    class ml_switcheroo.core.graph.GraphExtractor[source]Â¶

    Bases: libcst.CSTVisitor

    LibCST Visitor that extracts a LogicalGraph from Python source code.

    Two-Pass Logic:

    1. Init Pass: Scans __init__ or setup to register named layers assigned to self. Populates the node registry.

    2. Forward Pass: Scans forward or __call__ to trace variable usage. Builds edges between registered nodes based on data flow.
