ImageDecoder
============

Loads and decodes and image from a file. If it can't decode for any reason (e.g. corrupted encoded stream, invalid format, it will return an empty matrix). The following image formats are supported: * BMP * JPEG (note: Lossless JPEG support is optional) * JPEG2000 * TIFF * PNG * WebP * Portable imag...

**Abstract Signature:**

``ImageDecoder(encoded_stream, pixel_format: str)``

*No implementations mapped.*
