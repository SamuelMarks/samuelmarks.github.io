.. py:class:: TorchAdapter

   Adapter for PyTorch.

   Handles mapping of:
   1.  **Neural Networks**: `torch.nn.*` layers and containers.
   2.  **Math**: `torch.*` tensor operations.
   3.  **Types**: `torch.float32`, etc.
   4.  **Casting**: `.float()`, `.long()`, etc. mapped to Abstract Casts.
   5.  **Optimization**: `torch.optim.*`.
