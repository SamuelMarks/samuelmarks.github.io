    ml_switcheroo.core.rewriter.passes.apiÂ¶

    API Logic Pass.

    This module consolidates all API-level transformations, including: 1. Function Calls: Remapping APIs (e.g., torch.abs -> jnp.abs), applying argument normalization,

    handling layout permutations, and executing transformation strategies (infix, lambda, macros).

    1. Attributes: Remapping attributes/constants (e.g., torch.float32 -> jnp.float32).

    2. Assignments: Unwrapping functional return patterns (e.g., layer.apply).

    3. Symbol Resolution: Resolving aliases to fully qualified names for lookup.

    4. Scoping: Tracking stateful variables (layers) to inform call rewriting logic.
