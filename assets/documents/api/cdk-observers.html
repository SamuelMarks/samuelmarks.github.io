API reference for Angular CDK observers

import {ObserversModule} from '@angular/cdk/observers';

Directives

CdkObserveContent

Directive that triggers a callback whenever the content of its associated element has changed.

Selector: [cdkObserveContent]

Exported as: cdkObserveContent
Properties
NameDescription
@Input()

debounce: number

Debounce interval for emitting the changes.

@Output('cdkObserveContent')

event: new EventEmitter<MutationRecord[]>()

Event emitted for each change in the element's content.
