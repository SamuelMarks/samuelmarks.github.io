# IREE Compiler and Runtime (`iree-compiler`, `iree-runtime`)

ONNX9000 integrates the state-of-the-art IREE compiler toolchain and virtual machine execution engine directly into the browser.

Through `@onnx9000/iree-compiler`, ONNX graphs are lowered entirely in JavaScript/WebAssembly through the MLIR progression (ONNX -> MHLO -> Linalg -> HAL -> VM Bytecode).

Through `@onnx9000/iree-runtime`, the generated bytecode (.wvm) is interpreted or passed directly to WebGPU/WebNN for zero-overhead, highly optimized inference.

## Command Line Interface (CLI)

Both the Python and JS CLI allow you to orchestrate the IREE workflow offline:

```bash
# Compile an ONNX model to an IREE WVM binary
onnx9000 iree compile my_model.onnx

# Execute an IREE WVM binary
onnx9000 iree run my_model.wvm
```

## Python SDK

In Python, the IREE workflow handles the same transformations using the core `codegen` and compiler pipelines:

```python
from onnx9000_cli.main import iree_cmd
import argparse

# Alternatively, you can use the internal SDK functions
# that match the CLI workflow directly.
args = argparse.Namespace(iree_command="compile", model="model.onnx")
iree_cmd(args)
```

## JavaScript/TypeScript SDK

In the browser or Node.js environment, compilation and execution run natively:

```typescript
import { compileModel } from '@onnx9000/iree-compiler/src/cli.js';
import { Module, Context, WVMInterpreter, HALBindings } from '@onnx9000/iree-runtime/src/vm.js';
import * as fs from 'fs';

// 1. Compile AOT
await compileModel('my_model.onnx', {
  targetBackend: 'wasm',
  dumpMlir: false,
  optimizeLevel: 'O3',
});

// Assume the compiler emitted a "my_model.wvm" binary
const bytecode = new Uint8Array(fs.readFileSync('my_model.wvm'));

// 2. Initialize Runtime Context
const mod = new Module();
const ctx = new Context(mod);

// 3. Bind the Hardware Abstraction Layer
HALBindings.register(ctx, { mockDevice: false }); // Bind to WebGPU or real device

// 4. Run the Interpreter
const interpreter = new WVMInterpreter(bytecode, ctx);
await interpreter.runAsync();
```

## Interactive Web Demo

To interactively compile an ONNX model into Web VM bytecode and execute it locally:

```bash
onnx9000 serve
```

And navigate to `/iree` in your web browser.
