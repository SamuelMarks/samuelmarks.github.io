.. py:function:: evaluate_dispatch_rules(rewriter, node: libcst.Call, rules: List[Any], details: Dict[str, Any]) -> Optional[str]

   Evaluates conditional dispatch rules against the current call arguments.

   :param rewriter: The calling transformer object.
   :param node: The Call node from libcst being evaluated.
   :param rules: A list of conditional rules to evaluate.
   :param details: A dictionary containing details for rule evaluation, including variants.

   :returns: The string identifier of the API to dispatch to if a rule matches, otherwise None.
