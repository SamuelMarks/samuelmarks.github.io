tf2onnx Replication & Parity Tracker¶

Description¶

This document tracks the complete reimplementation of tf2onnx within the onnx9000 ecosystem. Unlike the original project which relies heavily on a massive native tensorflow installation to parse and trace graphs, this implementation uses a pure-Python, zero-dependency protobuf / flatbuffers parser. This allows the converter to run entirely in the browser via WASM/WebGPU or incredibly efficiently in serverless/distributed environments without pulling in gigabytes of TF dependencies.
