    Structural Validation Linter (The Anti-Pollution Check).

    This module provides the StructuralLinter, a static analysis tool designed to verify that the output of a transpilation contains no artifacts from the source framework.

    Detection Scope: 1. Direct Imports: Flags import torch or from flax import ‚Ä¶. 2. Aliased Usage: Tracks import torch as t and flags subsequent calls like t.abs(). 3. Attribute Access: Flags torch.nn.Linear if torch is a forbidden root.
