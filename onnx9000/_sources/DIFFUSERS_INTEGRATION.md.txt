# Diffusers Web Integration (`diffusers`)

ONNX9000 supports web-native execution of Stable Diffusion and other text-to-image models natively via `@onnx9000/diffusers`.

## CLI Usage

```bash
onnx9000 diffusers my_model --prompt "A lovely cat" -o output.png
```

## Interactive Web Demo

Run `onnx9000 serve` and navigate to `/diffusers`.
